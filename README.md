# 🎨 SEO Content Factory

> A sua fábrica de conteúdo com IA: gera **imagens** (fal.ai) e **vídeos** (Higgsfield) a
> partir das suas squads de conteúdo, com um **dashboard de aprovação** no seu Cloudflare —
> o cliente aprova antes de publicar, e a publicação sai sozinha.

Template — gere a sua cópia e instale com o Claude Code, sem programar.

## 🚀 Como começar (3 passos)

1. **`Use this template`** → `Create a new repository`.
2. **Baixe:** `git clone` + `cd`. (Windows? [Tutorial Windows](docs/windows.md).)
3. **Abra o Claude Code e diga `instalar`** (`claude` → **`instalar`**).
   O Claude configura a fal.ai, o Higgsfield e sobe o dashboard. Aula em [`aula/roteiro.md`](aula/roteiro.md).

## ✅ Pré-requisitos

- A **Base da Agência de IA** instalada (as skills de geração vêm dela).
- Conta **fal.ai** (API key) — geração de imagens.
- Plano **Higgsfield** ativo — geração de vídeos.
- Conta **Cloudflare** + **Node** + `gh` (pro dashboard de aprovação).

## 🧩 O que você ganha

- **Imagens** no padrão da sua marca (carrosséis, ads, posts) via fal.ai.
- **Vídeos** (cinematic, reels) via Higgsfield.
- **Dashboard de aprovação** embedável no CRM Funnels: o cliente vê o preview e aprova; a publicação
  vai pras redes automaticamente.
